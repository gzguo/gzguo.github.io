java.util.GregorianCalendar：是Calendar的子类，公立计算

开始学习java.util.Date:

  • 在标准的java类库中包含了一个Date类。他的对象表示特定的瞬间，精确到毫秒
  • java中时间表示的说白了也是数字，是从：标准纪元1970.1.1 0点开始到某个时刻的毫秒数，类型是long
